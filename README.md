# Academic Voting Poll System

## Introduction
This is a web project for an academic voting poll system. It helps users add poll options, vote for their preferred option, and view live voting results through vote counts and progress bars. It is built using React, Tailwind CSS, and JavaScript.

## Features
- Add poll options
- Vote once per user
- Live vote count
- Progress bar results
- Reset votes
- Responsive design

## Installation
To run the project locally:

1. Clone the repository from GitHub:

```bash
git clone https://github.com/EzraKipyego/Voting_App.git
```
2. Navigate into the project folder:
```bash
cd Voting_App
```

3. Install dependencies:
- npm install 
- npm createvite@latest


4. Start the development server:
- npm run dev


## Usage
- Open the project in your browser
- Add a new poll option using the input field
- Click the vote button to vote
- View vote counts and progress bars
- Reset votes when needed
